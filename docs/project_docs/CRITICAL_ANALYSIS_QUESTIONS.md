## Backend Deep Dive

### 101. Проблема с sys.path манипуляциями
**ВОПРОС:** Почему в каждом файле есть `sys.path.insert(0, str(Path(__file__).parent.parent.parent / ".venv" / "Lib" / "site-packages"))`? Это ломает импорты и делает код зависимым от структуры папок.
