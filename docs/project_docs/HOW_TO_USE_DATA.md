## 🔬 ТАБЛИЦЫ СМЕЩЕНИЯ ПОЧЕК - ПОЛНЫЙ АНАЛИЗ

### 📋 Какие данные содержатся:

#### 1. **Координаты почек** (X, Y, Z в мм)
- **Верхняя треть**: верхний полюс почки
- **Средняя треть**: центральная часть, ворота почки  
- **Нижняя треть**: нижний полюс
- **Положения**: на спине / на боку

#### 2. **Углы ротации почки**
- **∠А°** - угол в горизонтальной плоскости
- **∠В°** - угол в сагиттальной плоскости  
- **∠С°** - угол во фронтальной плоскости

#### 3. **Сосудистые параметры**
- **L (мм)** - длина сосудистой ножки
- **∠Отхождения°** - угол отхождения артерии
- **∠Проваливания°** - угол проваливания вены
- **Толщина паранефральной клетчатки** - спереди/сзади/сбоку

#### 4. **Индекс извитости**
- Оценка сложности доступа к сосудам
- Влияет на выбор троакара

#### 5. **Разница показателей** (Δ)
- **Смещение в мм** - насколько смещается почка
- **Смещение в %** - процентное изменение
- **Ключевые метрики** для предсказания
