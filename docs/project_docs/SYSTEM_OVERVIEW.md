# 🏥 AR LAPAROSCOPY - СИСТЕМНЫЙ ОБЗОР ДЛЯ ПРЕЗЕНТАЦИИ

## 📋 КРАТКОЕ ОПИСАНИЕ ПРОЕКТА

**Laprascope** — это медицинская AR-система для лапароскопической хирургии почек, которая позволяет хирургам визуализировать 3D-модели органов в дополненной реальности во время операций.

### 🎯 Основная цель
Повышение точности и безопасности лапароскопических операций на почках через AR-визуализацию анатомических структур.

---

## 🏗️ АРХИТЕКТУРА СИСТЕМЫ

### 4 основных модуля:
```
UI-Клиент (Unity) ←→ Серверная часть (Python/FastAPI) ←→ ML-модуль (TotalSegmentator)
                                    ↓
                              AR-модуль (внутри Unity)
```

### Технологический стек:
- **Backend**: Python + FastAPI + TotalSegmentator + PyTorch
- **Frontend**: React + TypeScript + Vite  
- **AR Client**: Unity 2022.3 LTS + ARFoundation + C#
- **3D Processing**: scikit-image + trimesh + marching cubes

---

## 🔄 КАК РАБОТАЕТ СИСТЕМА

### 1. Загрузка данных
- Хирург загружает DICOM файлы КТ-исследования
- Система автоматически выбирает правильную серию сканирования
- Поддержка ZIP-архивов и NIfTI форматов

### 2. AI-сегментация
- **TotalSegmentator** выделяет почки, сосуды, опухоли
- 104 органа автоматически сегментируются за 2-5 минут
- Валидация качества сегментации

### 3. 3D-генерация
- **Marching Cubes** алгоритм создаёт 3D-модели из медицинских изображений
- Оптимизация mesh (упрощение, сглаживание)
- Экспорт в STL формат для Unity

### 4. AR-визуализация
- Загрузка STL моделей в Unity
- Регистрация по костным структурам и ArUco маркерам
- Отображение в дополненной реальности на планшете

## 🎯 КЛЮЧЕВЫЕ ФУНКЦИИ ДЛЯ ХИРУРГОВ

### Интраоперационный режим (метод Федорцова)
- **Визуализация ЧЛС**: чашечно-лоханочная система
- **Анализ сосудов**: артерии и вены с цветовой кодировкой
- **Определение камней**: автоматическое выделение
- **Расчёт углов доступа**: зелёный/красные индикаторы безопасности
- **Измерительные инструменты**: тап-тап расстояния, углы, глубина

### Предоперационный режим (метод Коновалова)
- **3D проекция опухоли**: на брюшную стенку с топографией
- **Расчёт троакаров**: оптимальные точки доступа
- **Планирование доступа**: визуализация коридоров
- **Оценка рисков**: расстояние до критических структур

### Интеллектуальные функции
- **AI-ассистент**: рекомендации на основе анатомии
- **Предсказание смещения**: на основе реальных данных пациентов
- **Автоматическая сегментация**: опухоли, сосудов, ЧЛС
- **4D визуализация**: дыхательные движения
